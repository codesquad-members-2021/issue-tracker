## 새롭게 도전한 점
1. `Kakao REST API` + `Github` 으로 OAuth 구현 ⭐️

3. `myUseCallback()` hook 직접 구현 ⭐️
4. markdown으로 작성 가능한 Editor 직접 구현 ⭐️
